# Myntra Pants Product Analysis

## Objective
Analyze Myntra pants product data to understand pricing, discounts, and brand performance.

## Tools Used
Python, Pandas, NumPy, Matplotlib, Seaborn, Google Colab

## Key Insights
- Budget brands provide higher discounts
- Most pants fall in the ₹500–₹2000 price range
- Brand trust impacts ratings more than discounts
